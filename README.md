# 接口设计

> API接口设计并没有遵循RESTful，私以为RESTful能表示的语义有限，太多的特殊语义不好表示（我太菜了）。
>
> API接口的设计参照的是Github上Germey的项目[Gerapy](https://github.com/Gerapy/Gerapy/blob/master/gerapy/server/core/urls.py)，大致的风格是：`api/resource/operation`
